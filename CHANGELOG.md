# Changelog

All notable changes to this project will be documented in this file.

The project follows Semantic Versioning.

## [0.1.0] - 2026-07-12

### Added
- IPv4 and IPv6 OpenSSH authentication-event parsing.
- Repeated-failure and password-spray-style detections.
- JSON and Markdown reports.
- Evidence-based MITRE ATT&CK suggestions.
- Configurable detection thresholds and explicit limitations.
- Unit and CLI tests across Python 3.10, 3.11, and 3.12.
- Installable command-line entry point.
