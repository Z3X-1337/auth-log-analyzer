# Roadmap

## v0.2.0
- Add time-window-aware detections.
- Parse additional OpenSSH event formats.
- Add CSV export and richer evidence records.
- Add representative false-positive datasets.

## v0.3.0
- Add installable configuration profiles.
- Add batch and directory processing.
- Add Sigma-inspired rule metadata.
- Add optional enrichment interfaces without changing deterministic defaults.

## Non-goals
- Replacing a SIEM or EDR.
- Claiming attacker intent from threshold matches alone.
- Processing private production logs in public demonstrations.
